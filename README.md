# Setup
```
 1 - composer install
 
 2 - php bin/console doctrine:database:create 
 
 3 - php bin/console doctrine:migrations:migrate --no-interaction
 
 4 - php bin/console doctrine:fixtures:load
  
 5 - php bin/console server:run
```
